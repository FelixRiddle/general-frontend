# General frontend


## Getting Started

Clone 'good-roots-ts-api'
Cd to it and run:

```bash
npm run build && npm link
```

Link will make this package usable on the local environment

Now in 'general-frontend' run:

```bash
npm link good-roots-ts-api
```

And then run the development server:

```bash
npm run dev
```
